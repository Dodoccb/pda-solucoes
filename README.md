# Site PDA Soluções

Site simples, moderno e responsivo em HTML + CSS puro para publicação no GitHub Pages.

## Estrutura

- `index.html` — estrutura completa do site
- `style.css` — estilos organizados por seções
- `assets/logo-pda.png` — logo da PDA
- `assets/appda-dashboard.png` — imagem/mockup do APPDA usada no hero e seção APPDA

## Antes de publicar

1. Trocar o número `5541998226732` pelo WhatsApp oficial da PDA.
2. Confirmar se o repositório do GitHub Pages está apontando para o domínio `pdasolucoe.com.br`.
3. Depois de publicar, configurar o domínio no GitHub Pages.
4. Criar os arquivos `robots.txt` e `sitemap.xml` quando o site estiver no ar.
5. Cadastrar o domínio no Google Search Console.
